Modernization map

Modern VBScript syntax

Use UDTs, enums, static variables, optional parameters, ParamArray, explicit ByVal, non-zero array bounds, and type declarations via As Type.

JavaScript for new modules

Build with ES6+ features such as classes, promises, async/await, destructuring, template literals, and module imports.

AxonLive reactive engine

Handle events asynchronously and patch only changed DOM components instead of reloading full pages.

AI-ready MCP integration

Connect AI agents to your runtime context and generate ASP pages using available native objects and functions.

Container and edge deployment

Deploy quickly with Docker or existing web infrastructure using reverse proxy or FastCGI mode.

Operational confidence

Use built-in test tooling, centralized configuration, and deterministic runtime behavior for production pipelines.

Build path in four stages

1. Keep existing ASP pages online

Start by running current applications with AxonASP and preserve existing behavior.

2. Add new language-level features

Introduce modern VBScript options and ES6+ JavaScript where they add immediate value.

3. Split code into modules

Use partial CommonJS-like and ES6-style imports to organize JavaScript logic for scale.

4. Deliver reactive UX with AxonLive

Convert high-interaction pages to server-driven reactive components with targeted updates.
