# G3AXON.LIVE Properties

The following properties are available for the `G3AXON.LIVE` object.

| Property | Access | Type | Description |
|---|---|---|---|
| [EventArgs](properties/eventargs.md) | Read-only | String | Returns the entire event arguments map encoded as a JSON string. |
| [EventComponentID](properties/eventcomponentid.md) | Read-only | String | Returns the ID of the component that fired the asynchronous event. |
| [EventName](properties/eventname.md) | Read-only | String | Returns the name of the asynchronous event that was fired (e.g., `onclick`). |
| [IsAsyncRequest](properties/isasyncrequest.md) | Read-only | Boolean | Returns `True` if the current request is an asynchronous G3AxonLive event POST, otherwise `False`. |
| [Version](properties/version.md) | Read-only | String | Returns the current version of the AxonLive framework (e.g., `2.0.0`). |
